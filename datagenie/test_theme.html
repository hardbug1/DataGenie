📊 분석 결과

매출 증가율: 지난 3개월간 15.2% 증가했습니다.

  • 1월: 125,000원
  • 2월: 142,300원
  • 3월: 158,750원

💡 권장사항: 현재 증가 추세를 유지하기 위한 마케팅 전략을 강화하세요.

🔍 상세 데이터

이 텍스트는 다크 모드에서는 흰색으로, 라이트 모드에서는 검은색으로 표시됩니다.

SELECT * FROM sales WHERE date >= '2024-01-01'

📊 차트 데이터 예시

차트 제목: 월별 매출 현황

X축: 1월, 2월, 3월, 4월

Y축: 125K, 142K, 158K, 175K

범례: 매출액(만원)

※ 실제 Plotly 차트에서 모든 텍스트 요소(제목, 축 레이블, 범례, 데이터 레이블)가 테마에 맞게 자동 조정됩니다.

✨ 개선된 테마 기능

• 자동 감지: 시스템 설정에 따라 자동으로 다크/라이트 모드 전환

• 고대비: 밝은 배경에는 검은 글씨, 어두운 배경에는 흰 글씨

• 차트 최적화: Plotly 차트의 모든 텍스트 요소 색상 자동 조정

• 실시간 적용: 테마 변경시 즉시 반영

• JavaScript 감지: 배경 밝기 실시간 분석 및 최적 대비 적용
